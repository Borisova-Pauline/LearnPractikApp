## Название
Мои Коллекции
<br>

## Описание
Данный продукт был разработан в рамках учебной практики. Программа позволяет пользователям вести учёт их коллекций и всегда иметь под рукой свои записи с возможностью импорта из уже готовых файлов или экспорта данных коллекции. 
<br>

## Установка
Для установки программы нужно зайти в раздел `releases` и установить `.apk` из самого нового релиза, после чего найти скачанный файл в своём файловом менеджере и нажать на него. В открывшемся окне подтвердить установку 
После завершения процесса окно закроется и приложение будет установлено на устройство.
<br>

## Использование
Данное приложение позволяет работать с коллекциями, внутри которых находятся таблицы, что даёт инструмент ведения учёта.
При создании коллекции есть выбор: вручную и импорт из .XLSX файла. При создании вручную пользователь выбирает количество столбцов и строк.<br>
Внешний вид таблицы полностью зависит от пользователя. В приложении есть возможность создания, редактирования и удаления столбцов и строк<br>
Содержимое ячеек на данный момент ограничивается только текстом, без возможности вставлять картинки<br>
Приложение позволяет экспортировать таблицы в .XLSX файлы
<br>
